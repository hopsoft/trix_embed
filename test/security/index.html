🔒 Security

Trix Embed ships with a few security features to help prevent malicious and unwanted content from being embedded.

All security attributes are removed from the DOM when the controller connects, and all security values are discarded when the controller disconnects.

🗂️ Files

Traditional file attachments by any means are disabled. This includes pasting, dragging, and dropping.

✅ Allow List

Allow lists let you specify which hosts and domains can be embedded. The lists are encrypted and hidden making it difficult for attackers to tamper with them. 

data-trix-embed-list-value='["example.com", "test.com"]'

😱 Paranoid

Paranoid mode is enabled by default.

The paranoid option protects your forms from being tampered with. Read more about the attack vectors this protects against. 

data-trix-embed-paranoid-value="true"
