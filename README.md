## Developing

```sh
git clone https://github.com/hopsoft/trix_embed.git
cd trix_embed
yarn
yarn build
yarn start
```

## Releasing

1. Run `yarn` to pick up the latest
1. Run `yarn build`
1. Run `yarn publish`
1. Yarn will prompt you for the new version. Pre-release versions use `-preN`
1. Commit and push changes to GitHub
